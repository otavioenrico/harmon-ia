Perguntas frequentes

Quando lança?

Estamos em desenvolvimento. Quem estiver na lista de espera é avisado primeiro, antes do público geral.

Preciso de cartão de crédito para entrar na lista de espera?

Não — é só o seu e-mail. Nada de pagamento nesta fase.

Qual a diferença entre Personal e Team?

Personal é para quem atende sozinha. Team é para clínicas com mais de uma profissional usando a mesma conta.

Dá para usar em qualquer dispositivo?

Sim — é 100% web, funciona em qualquer navegador, sem instalar nada.

Meus dados ficam seguros?

Sim — cada conta só acessa os próprios dados, com login pelo Google.
